# Socket FTP Client 前端界面操作说明

本文档说明图形界面的主要区域和常用操作流程。新版界面采用类似 WinSCP Commander 的双栏文件管理思路：左侧是本地文件，右侧是远端服务器，连接配置放在独立的“站点管理”窗口中，避免把所有功能挤在一个界面里。

## 1. 界面区域

- 菜单栏：提供“会话”“文件”“查看”三个入口，适合查找完整功能。
- 工具栏：放置高频操作，包括站点管理、连接、断开、上传、下载、刷新、上级、新建目录、重命名、删除、属性、切换主题。
- 左侧本地文件面板：以表格方式浏览本机目录，选择要上传的文件。第一行 `..` 表示返回父目录。
- 右侧远端服务器面板：以表格方式浏览 FTP 服务器目录，查看文件名称、类型、大小、修改时间、权限。第一行 `..` 表示返回父目录。
- 站点管理窗口：独立管理 FTP 连接信息，不占用日常文件管理空间。
- 底部传输队列：查看上传/下载任务的状态、进度、速度和错误信息。
- 底部协议日志：显示真实 FTP 命令和服务器响应，可用于说明程序确实基于 socket 手写 FTP 交互。

## 2. 连接 FTP 服务器

1. 点击工具栏或“会话”菜单中的“站点管理”。
2. 在弹出的站点管理窗口中填写主机，例如 `test.rebex.net`。
3. 端口通常填写 `21`。
4. 填写用户名和密码；匿名服务器可使用 `anonymous`。
5. 点击“连接”。
6. 连接成功后，站点管理窗口会关闭，工具栏右侧状态变为“已连接”，右侧远端列表会显示服务器当前目录。

示例：

```text
站点名：Rebex 测试服务器
主机：test.rebex.net
端口：21
用户：demo
密码：password
```

## 3. 保存和复用站点

1. 打开“站点管理”。
2. 填写连接信息。
3. 如需保存密码，勾选“保存密码”；默认不保存明文密码。
4. 点击“保存站点”。
5. 下次打开程序后，在站点管理窗口左侧双击该站点，即可自动回填主机、端口和用户名。

## 4. 浏览远端目录

- 双击远端列表中的目录，可进入该目录。
- 双击远端列表顶部的 `..`，可返回父目录。
- 点击“上级”可返回父目录。
- 点击“刷新”可重新读取当前远端目录。
- 也可以在“远端路径”输入框中输入路径，例如 `/pub/example`，然后点击“打开”。
连接、刷新、进入目录等网络操作会在后台执行，界面底部会显示“正在读取远端目录”等状态提示。

## 5. 上传文件

1. 确认已经连接服务器。
2. 在左侧本地文件面板选中要上传的文件。
3. 在右侧远端区域进入目标目录。
4. 点击工具栏的“上传”。
5. 上传任务会出现在底部“传输队列”中。

如果没有在左侧选中文件，程序会弹出文件选择窗口，允许临时选择一个文件上传。

也可以直接把左侧本地文件拖到右侧远端面板，程序会把该文件上传到当前远端目录。

断点续传说明：如果远端已有同名的部分文件，程序会尝试使用 `SIZE` 获取远端大小，并通过 `REST` + `STOR` 从断点继续上传。服务器不支持 `REST` 时，协议日志会提示并退回完整覆盖上传。

## 6. 下载文件

1. 在左侧本地文件面板进入目标保存目录。
2. 在右侧远端列表中选中一个文件。
3. 点击“下载”。
4. 文件会下载到左侧当前本地目录，任务会出现在底部“传输队列”中。

也可以直接把右侧远端文件拖到左侧本地面板，程序会把该文件下载到左侧当前目录。

断点续传说明：如果本地已经存在同名的部分文件，程序会读取本地文件大小，并通过 `REST` + `RETR` 从断点继续下载。下载完成后可通过文件大小或哈希校验确认结果。

## 7. 右键菜单

左侧本地文件面板支持右键菜单：

- 打开：目录会进入该目录，文件会显示属性。
- 上传到远端：上传当前选中的本地文件到右侧远端当前目录。
- 新建目录：在左侧当前目录新建文件夹。
- 重命名：重命名本地文件或目录。
- 删除：删除本地文件；目录需要为空。
- 属性：查看本地路径、类型、大小、修改时间。
- 刷新：重新读取当前本地目录。

右侧远端服务器面板支持右键菜单：

- 打开：进入远端目录；顶部 `..` 表示返回父目录。
- 下载到左侧：下载远端文件到左侧当前本地目录。
- 新建目录：在远端当前目录新建文件夹。
- 重命名：重命名远端文件或目录。
- 删除：删除远端文件；目录通常需要为空。
- 属性：查看远端类型、大小、修改时间、权限和原始列表行。
- 刷新：重新读取远端目录。

## 8. 暂停、继续、取消传输

- 在底部“传输队列”选中任务。
- 点击“暂停”会中断当前任务并保留已传输部分。
- 点击“继续 / 重试”会重新入队，并基于断点续传逻辑继续。
- 点击“取消”会停止任务。

## 9. 远端文件管理

- 新建目录：点击“新建目录”，输入目录名。
- 重命名：选中远端文件或目录，点击“重命名”，输入新名称。
- 删除：选中远端文件或目录，点击“删除”并确认。
- 属性：选中远端项目，点击“属性”，查看类型、大小、修改时间、权限和原始列表行。

## 10. 查看协议日志

切换到底部“协议日志”页签，可以看到类似下面的命令和响应：

```text
C USER demo
S 331 Password required
C PASS ********
S 230 User logged in
C EPSV
S 229 Entering Extended Passive Mode
C REST 102400
S 350 Restarting at 102400
C RETR big-file.zip
S 150 Opening binary mode data connection
S 226 Transfer complete
```

其中 `C` 表示客户端发出的 FTP 命令，`S` 表示服务器响应。密码会在日志中自动脱敏。

## 11. 中文显示排查

程序启动时会优先选择以下中文字体：微软雅黑、Noto Sans CJK SC、思源黑体、文泉驿微米黑、黑体、宋体等。

Windows 环境通常自带微软雅黑，因此中文应正常显示。WSL/Linux 如果界面中文显示为方块，说明系统没有安装 CJK 字体。Ubuntu/WSL 可执行：

```bash
sudo apt install fonts-noto-cjk
```

安装后重新运行：

```bash
bash scripts/run.sh
```
